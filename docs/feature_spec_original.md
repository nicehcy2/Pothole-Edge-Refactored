# 기능 명세서 (원본)

## 1. GPS 수신
- 외부 기기(앱, GPS 모듈)로부터 위경도 + 속도를 POST로 수신
- 수신한 GPS 데이터를 전역 변수로 유지, 감지 루프에서 주기적으로 참조

## 2. 포트홀 감지
- 웹캠 프레임을 YOLOv8 모델에 입력해 포트홀 감지
- 신뢰도 임계값 이상인 결과만 처리
- 감지된 프레임을 지오해시 기반 폴더 구조로 로컬 저장

## 3. 중복 감지 방지
- 감지 위치의 지오해시를 백엔드 API로 조회
- 이미 등록된 지오해시면 S3 업로드 및 DB 등록 스킵

## 4. 최적 이미지 선택
- 같은 지오해시 폴더 내 여러 프레임 중 신뢰도 가장 높은 이미지 1장만 선택
- 나머지는 삭제

## 5. S3 업로드
- 백엔드에서 Presigned URL 발급받아 이미지 PUT 업로드

## 6. 포트홀 등록
- S3 업로드 완료 후 백엔드에 위경도 + 이미지 URL + 지오해시 등록

## 7. 비동기 업로드
- 감지 이미지가 N장 쌓이면 별도 스레드로 업로드 처리
- 감지 루프를 블로킹하지 않음
